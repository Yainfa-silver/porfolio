# PORTFOLIO.SYS // CRT-AESTHETIC

Un espacio digital personal con una estética profundamente inspirada en la cultura visual Y2K, el cyberpunk clásico y la serie **Serial Experiments Lain**.

## 🖥️ Resumen del Proyecto

Este portfolio es una experiencia web estática que simula la interfaz de un terminal antiguo o un monitor CRT de finales de los 90. Utiliza efectos visuales dinámicos para sumergir al visitante en una atmósfera nostálgica y tecnológica.

### ✨ Características Principales

- **Efectos CRT Reales**: Implementación de *scanlines*, curvatura de pantalla, viñeta y ruido estático para simular un monitor analógico.
- **Interfaz "Filesystem"**: Un sistema de navegación rápida inspirado en menús de sistemas operativos retro para cambiar cómodamente entre secciones.
- **Interactividad**: Un personaje animado que reacciona y ofrece diálogos dinámicos al interactuar.
- **Sistema de Canales**: La sección "Sobre mí" utiliza un sistema de navegación por pestañas/canales que imita el cambio de sintonía en una televisión antigua.
- **Responsive**: Adaptado para mantener la legibilidad y el estilo visual tanto en dispositivos móviles como en escritorio.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: HTML5 y CSS3 (Vanilla).
- **Lógica**: JavaScript puro para efectos de glitch, navegación y diálogos.
- **Iconografía**: Lucide Icons y Simple Icons.
- **Tipografía**: Fuentes monoespaciadas retro (Share Tech Mono, VT323).

## 📁 Estructura del Sistema

- `index.html`: Terminal principal y centro de control.
- `sobre-mi.html`: Biografía y detalles estructurados por canales.
- `proyectos.html`: Repositorio de proyectos y experimentos.
- `css/main.css`: El núcleo del diseño y los efectos visuales.
- `js/`: Lógica modular para el menú, diálogos y efectos visuales.

## 🚀 Cómo Visualizar

Para ejecutar el sistema localmente:
1. Clona o descarga el repositorio.
2. Abre `index.html` en cualquier navegador moderno.
